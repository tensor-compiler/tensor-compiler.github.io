Format Functions¶

pytaco.is_dense(fmt)¶

Checks if a format is all dense.

Parameters
fmt: pytaco.format
Returns
bool

True of all dimensions (modes) in a tensor are stored in a dense format and False otherwise.

Examples

>>> import pytaco as pt
>>> pt.is_dense(pt.csr)
False
>>> my_fmt = pt.format([pt.dense]*3)
>>> pt.is_dense(my_fmt)
True
